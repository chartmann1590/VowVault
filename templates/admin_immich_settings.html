How It Works

Immich Sync Process:
  1. Photos/videos are uploaded to the wedding gallery
  2. System automatically syncs to your Immich server
  3. Files are added to the specified album
  4. Descriptions include uploader name and details
  5. Sync status is logged for troubleshooting
What Gets Synced:
  • All uploaded photos and videos
  • Guestbook photo attachments
  • Message board photo attachments
  • Photobooth photos
  • Original file names preserved
Setup Notes:
  • Ensure your Immich server is accessible
  • Generate an API key in Immich admin settings
  • Test the connection before enabling sync
  • Monitor sync logs for any issues
